# Nested Showcase

**Created:** 2025-11-17  
**Updated:** 2025-11-17

## Description
This parent todo exists only to demonstrate how nested todos appear inside the `todo_mgr`
interactive UI. It should always have at least one child so the Kanban board shows
relationships.

## Requirements
- Parent stays in `Needs_Derivation` until children are complete
- Children inherit tags/flags where helpful

## Dependencies
**Parent todo:** none  
**Depends on:** none  
**Related:** demo coverage for todo_mgr documentation

## Outputs
- Reference screenshots showing nested todos
- Example AppleScript demo linking to nested structure
